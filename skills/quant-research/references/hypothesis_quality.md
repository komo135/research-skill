# hypothesis_quality.md (RETIRED)

The 4-gate hypothesis quality system has been subsumed into workstream-specific
planning and review:

- R&D Workstream: `rd_plan.md` (R&D category, uncertainty, method, evidence,
  decision criteria, and report-package comparison discipline)
- Phenomenon / mechanism research: `references/pure_research/pr_workflow.md`
  (discriminating trial loop, deviation severity matrix, push analysis depth
  before new trial) and `references/pure_research/preregistration.md`
  (pre-reg discipline)

The "hypothesis" abstraction is not the shared top-level state object. R&D
uses the plan and report package as the unit of discipline; phenomenon /
mechanism research uses the explanation as the unit.

Do not consult this file for current protocol.
