# Deprecated Research Ideation Reference

This reference is deprecated. Research idea generation now uses `references/mechanistic_hypothesis_generation.md`.

Use the new protocol when the user asks for research ideas, research directions, hypothesis candidates, or "what should we try next." It starts with research situation diagnosis, chooses the hypothesis type, derives predictions or expected observations, and decides `commit / park / kill`. It uses a Mechanism hypothesis record only when the selected hypothesis is mechanistic.

This stub remains only so older links fail clearly instead of silently invoking the former candidate-generation workflow.
